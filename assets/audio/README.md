# Audio Assets

This directory should contain audio files for Mahjong Epic.

## Required Files

- `background.mp3` - Background music loop (30-60 seconds, royalty-free)
  - Recommended: Calm, relaxing music
  - Format: MP3, 128kbps or higher
  - Looping: Should be seamless for looping

## Notes

- The app handles missing audio files gracefully (no crash)
- Audio can be toggled on/off in Settings
- In development, audio may not play without a physical device or emulator
